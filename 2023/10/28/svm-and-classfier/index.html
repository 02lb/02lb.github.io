==核方法==

  • 核方法是一类在机器学习中常用的技术，用于处理非线性问题和高维数据。它通过将输入数据映射到一个高维特征空间，并在该特征空间中进行线性操作来解决非线性问题。

  • 核方法的目标是通过引入核函数，将原始数据从低维空间映射到更高维度的特征空间，从而使数据在新的特征空间中更容易分割。

  • 核函数是核方法的关键组成部分，它能够计算两个样本在高维特征空间中的内积，而无需显式地进行特征空间的映射。避免了直接计算高维特征空间的复杂性，并在计算上具有高效性。通过核函数 k(x, y)，我们就可以简化对偶形式，不必计算 ϕ 的内积：

  • 常见的核函数包括线性核函数、多项式核函数、高斯核函数等。每种核函数都有不同的属性和适用范围，可以根据具体问题选择合适的核函数。

==Hinge 损失==

$$
\begin{aligned}
L(\boldsymbol{w}, b) &=C \sum_{n=1}^{N} E_{S V}\left(y^{(n)} h^{(n)}\right)+\frac{1}{2}|\boldsymbol{w}|^{2} \
&=\sum_{n=1}^{N} E_{S V}\left(y^{(n)} h^{(n)}\right)+\lambda|\boldsymbol{w}|^{2}
\end{aligned}
$$

  • ​ 其中，$E_{SV}(z)=\max(0,1-z)$。

==Cross_entropy 损失==

$$
\begin{aligned}
L(w, b) &=-\sum_{n=1}^{N}\left[\tilde{y}^{(n)} \log \sigma\left(h^{(n)}\right)+\left(1-\tilde{y}^{(n)}\right)\left(1-\sigma\left(h^{(n)}\right)\right)\right]+\lambda|w|^{2} \
&=\sum_{n=1}^{N} \log \left(1+\exp \left(-y^{(n)} h^{(n)}\right)\right)+\lambda|w|^{2} \
&=\sum_{n=1}^{N} E_{L R}\left(y^{(n)} h^{(n)}\right)+\lambda|w|^{2}
\end{aligned}
$$

  • 其中，$E_{LR}(z)=\log(1+\exp(-z))$，$\tilde y\in{0,1}$，$y\in{-1.1}$

==采⽤ hinge loss 的线性分类模型和 SVM 模型之间的关系==

  • 采用hinge损失的软线性最大间隔分类器的损失函数如下：

$$
\begin{aligned}
L(\boldsymbol{w}, b) &=\sum_{n=1}^{N} max\left(0, 1-y^{(n)} h^{(n)}\right)+\lambda|\boldsymbol{w}|^{2}
\end{aligned}
$$

  • 其对应的对偶问题为：

$$
\begin{array}{l}
\max {\boldsymbol{a}} g(\boldsymbol{a})\
\text { s.t.: } a{n} \geq 0, a_{n} \leq C\
\quad\quad\sum_{n=1}^{N} a_{n} y^{(n)}=0
\end{array}
$$

  • 其中：
    $$
    g(\boldsymbol{a})=\sum_{n=1}^{N} a_{n}-\frac{1}{2} \sum_{n=1}^{N} \sum_{m=1}^{N} a_{n} a_{m} y^{(n)} y^{(m)} \boldsymbol{x}^{(n) T} \boldsymbol{x}^{(m)}
    $$

  • 使用了核方法的SVM就是在使用hinge损失的线形分类模型上进一步使用$\phi$函数把原始数据 $x$ 映射到更高维数的特征空间，来使其非线性化，即：

$$
g_{kernel}(\boldsymbol{a})={\sum_{n=1}^{N} a_{n}-\frac{1}{2} \sum_{n=1}^{N} \sum_{m=1}^{N} a_{n} a_{m} y^{(n)} y^{(m)} \phi\left(x^{(n)}\right)^{T} \phi\left(x^{(m)}\right)} = {\sum_{n=1}^{N} a_{n}-\frac{1}{2} \sum_{n=1}^{N} \sum_{m=1}^{N} a_{n} a_{m} y^{(n)} y^{(m)} k\left (\boldsymbol x^{(n)}, \boldsymbol{x} ^{(m)} \right )}
$$

def get_data():
    # 读入数据
    train = pd.read_csv('dataset/mnist_01_train.csv')
    test = pd.read_csv('dataset/mnist_01_test.csv')
    # 变换csv格式为np格式
    x_train = train.iloc[:,1:].values
    y_train = train.iloc[:,0].values
    x_test = test.iloc[:,1:].values
    y_test = test.iloc[:,0].values
    # 打乱数据
    num_samples = x_train.shape[0]
    random_index = np.random.permutation(num_samples)
    x_train = x_train[random_index]
    y_train = y_train[random_index]
    # normalization
    num_features = x_train.shape[1]
    mean_train = np.mean(x_train)
    mean_test = np.mean(x_test)
    std_train = np.std(x_train)
    std_test = np.std(x_test)
    x_train = (x_train-mean_train)/std_train
    x_test = (x_test-mean_test)/std_test
    return  x_train,y_train,x_test,y_test

==不同核函数的模型实现==

  • 定义获取模型的函数，核函数类型作为参数导入。
  • 调用sklearn.svm创建支持向量机的方式十分简单。
  • 注意对SVC传入参数类型作用：
    1. C=：惩罚项参数，控制了分类错误的惩罚程度。较小的C值会导致容忍更多的错误分类，较大的C值会更严格地对待错误分类，默认值为1.0。
    2. kernel=：用于指定支持向量机的核函数类型。常见的核函数包括线性核（linear）、多项式核（poly）、高斯径向基函数核（rbf）等。默认值为’rbf’。
    3. degree：多项式核函数的阶数。只有当kernel参数为’poly’时才需要设置。
    4. gamma：核函数系数。对于多项式核和高斯径向基函数核来说，gamma控制了决策边界的曲率。较小的gamma值会导致较平缓的决策边界，而较大的gamma值会导致决策边界更加复杂。默认值为’auto’，表示使用1/n_features。
    5. probability：是否启用概率估计功能。如果设置为True，训练时会为每个类别生成概率估计，默认值为False。
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score
def get_model(kernel_type):

    def gradient_descent(self, x_batch, y_batch):
        """
        根据不同损失函数进行梯度下降
        """
        if self.loss_type == 'hinge':
            gradient_W = np.dot(x_batch.T,
                (y_batch * (np.dot(x_batch, self.W)+self.b)<1).astype(int) * -y_batch) / self.batch_size
            gradient_b = np.mean((y_batch *
                                  (np.dot(x_batch, self.W) + self.b) < 1).astype(int) * -y_batch)
            self.W -= self.lr * gradient_W
            self.b -= self.lr * gradient_b

        if self.loss_type == 'cross_entropy':
            gradient_W = -np.dot(x_batch.T, 
                             y_batch / (1+np.exp(y_batch * (x_batch@self.W+self.b)))) / self.batch_size
            gradient_b = -np.mean(y_batch / (1+ np.exp(y_batch*(x_batch@self.W+self.b))))
            self.W -= self.lr * gradient_W
            self.b -= self.lr * gradient_b
  • 根据不同的损失类型，按照公式计算并记录损失值
    1. 输入指定了type参数，指示是训练集损失或者验证集损失。
    2. 对于损失求的是平均损失，使用了np.mean()，所以损失的绝对值会较小，损失值不会受到样本数量的影响。
    def loss(self, x_batch, y_batch, type):
        """
        计算并记录损失值
        :param type: 指定损失类型：训练集损失/验证集损失并记录
        """
        loss_num = None
        if self.loss_type == 'hinge':
            loss_num = np.mean(np.maximum(0, 1-y_batch*(x_batch@self.W+self.b)))
        if self.loss_type == 'cross_entropy':
            loss_num = np.mean(np.log(1 + np.exp(-y_batch*(x_batch@self.W+self.b))))
        if type == 'train':
            self.train_loss.append(loss_num)
        elif type == 'valid':
            self.valid_loss.append(loss_num)

def train(svm_model,x_train,y_train,x_test,y_test):
    time_start = time.time()
    svm_model.fit(x_train, y_train)
    time_end = time.time()
    # prediction
    predictions = svm_model.predict(x_test)
    # 统计性能
    time_consume = time_end-time_start
    accuracy = accuracy_score(y_test, predictions)
    precision = precision_score(y_test, predictions)
    recall = recall_score(y_test, predictions)
    # 统计支持向量的个数
    # supported_vector = svm_model.support_vectors_
    return time_consume, accuracy, precision, recall

==线形分类器超参数选择==

100 0.650714 0.999527 0.999119 1.0

  • 惩罚项参数C：

    1. 主要设置了三个阶段的参数：1,0.01,100
    2. 对照实验结果可以看出，由于mnist数据集本身具有明显的可分性，svm模型可能对C的取值不敏感。即使C的值变小或变大，模型的分类结果几乎没有区别而且分类效果较好。

  • 从运行时间对比：

    1. 高斯核函数训练用时稳定在0.65s左右，线形核函数训练用时稳定在0.36s左右。
    2. 由于高斯核函数相对线性核函数复杂，训练用时明显大于线形核函数。

  • 从模型性能对比：

    1. 从实验结果可以观察出，对于数据集两种核函数的准确率都很高。相对而言，高斯核函数在准确率和召回率上高了0.05%左右。
    2. 由于线形核函数是一个线性最大间隔分类器，对于线形不可分的数据性能较差。而高斯核函数能够将数据映射到更高维的特征空间，所以模型性能使用高斯核函数的SVM更优。
    3. 由于本次使用的mnist手写数字分类训练集本身比较简单，可分性较强，所以没有体现出两种模型很大的区别。
    4. 在召回率可以看出，存在某些正例样本使得线形核函数无法分类成功，而高斯核函数能够准确分类，达到了100%的召回率。

线形分类器损失函数类型 训练用时 测试集准确率

  • 两种线形分类模型的主要区别在损失函数的不同。hinge损失主要从最大间隔角度考虑，鼓励模型产生更大的间隔，以更好地分离不同类别的样本；而交叉熵损失主要从概率角度考虑，通过最大化正确类别的概率来优化模型。
  • 在本次实验中，由于数据集的可分性较强，数据类型比较简单，所以两种分类器都没有遇到困难，分类精度在99.9%以上。模型的训练用时也很接近。

==Section6 实验总结和心得体会==

==实验总结：==

模型类型 模型训练用时 最终测试集准确率
线性核函数SVM 0.353328 0.999054
高斯核函数SVM 0.645979 0.999527
hinge loss 线性分类器 4.481224 0.999054
cross-entropy loss 线性分类器 4.676503 0.999527

  • 这次实验总共实验了以上四种模型用于手写数字0/1图片集的简单分类问题。

  • 由于实验的数据集可分性很强，这四种模型最终收敛到了很高的准确度，即都有非常好的效果。

  • 从运行时间上看，由于两种线形分类器主要在计算梯度时有所区别外其它训练过程一致，所以用时接近。而两种核函数的SVM训练用时体现了出了差别，体现了高斯核函数较于线性核函数的复杂性。

  • 对于最终模型准确度雷同的问题，我认为是可能是原始数据集的标注有错误或者是无法区分类别，这些样本导致了模型只在这几个特例出错，最终准确率相同。

==心得体会：==

  • 本次实验的实现相对简单，SVM模型的原理以及实现老师已经在课堂上详细阐述过，所以实现起来比较容易。

  • 遇到的主要问题在于，对于损失函数的手动求导不熟悉，尤其是涉及到一些函数的偏导已经矩阵向量求导时无从下手。由于之前训练模型的时候依赖于backword()自动求导方法，所以还得多加训练。
